Формат конфигурационного файла для инструмента диагностики Alterator.

'''Таблица diag.1 Ключи базовой секции'''

{| class="wikitable" style="margin:auto"
|+ Caption text
|-
! '''Ключ''' !! '''Описание''' !! '''Тип значения''' !! '''Обязательный'''
|-
| Name || Уникальное имя сущности обозначенной в базовой секции диагностических инструментов. || string || Да
|-
| Type || Тип определят назначение файла конфигурации. Для диагностических инструментов это diag. || string || Да
|-
| Comment || Общее описание диагностических инструментов.  || string || Нет
|-
| DisplayName || Отображаемое имя в интерфейса Alterator. При отсутствии параметра отображается значение ключа Name. || string || Нет
|-
| DiagTask || Список уникальных имен(unic name) диагностических задач разделенных символом ';'. Каждый отдельный элемент списка раскрывается обозначенных ниже группах вида -- <code>DiagTask <unic-name></code>. || string || Да
|}

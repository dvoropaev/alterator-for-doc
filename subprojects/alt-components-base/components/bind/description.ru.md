Berkeley Internet Name Domain (BIND) — это программный комплекс, реализующий
сервер доменных имен в сети Интернет. BIND является наиболее широко используемым
программным обеспечением DNS-сервера в Интернете и поддерживается организацией
Internet Systems Consortium (ISC).

**Домашняя страница:** <https://bind.isc.org/>

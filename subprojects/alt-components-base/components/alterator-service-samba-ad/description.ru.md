Предоставляет централизованный сервис управления для развёртывания и администрирования Samba Active Directory через Альтератор.

Позволяет развёртывать и управлять контроллерами домена Samba AD со следующими возможностями:

* Развёртывание новых контроллеров домена Samba AD
* Удаление существующих контроллеров домена Samba AD
* Получение информации о параметрах развёрнутых контроллеров домена
* Добавление новых контроллеров домена в существующие домены
* Использование Samba-internal в качестве DNS бэкенда для развёртывания доменов

Provides centralized management service for Samba Active Directory deployment and administration through Alterator.

Allows deployment and management of Samba AD domain controllers with the following capabilities:

* Deploy new Samba AD domain controllers
* Remove existing Samba AD domain controllers
* Retrieve information about deployed domain controller parameters
* Add new domain controllers to existing domains
* Use Samba-internal DNS backend for domain deployment

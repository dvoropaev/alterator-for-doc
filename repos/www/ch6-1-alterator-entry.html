6.1. Каталог alterator-entry

Каталог repos/alterator-entry содержит спецификацию Alterator Entry, схемы и библиотечный код для работы с описаниями сущностей.Источник

Структурные элементы:

  • doc/README.md — основная спецификация по форматам, типам сущностей и правилам локализации.Источник
  • schemas/ — каталог схем для валидации формата.Источник
  • alterator_entry/ — исходный код Python-пакета.Источник
  • examples/ — примеры описаний для обучения и тестирования.Источник

Этот каталог служит точкой входа для изучения формата и подготовки собственных описаний.

5.2. Сборка ADT на ALT Linux

Для сборки ADT на дистрибутивах ALT Linux необходимо установить зависимости Qt5, CMake и Boost.Источник

Порядок действий:

  1. Установить зависимости командой:
    apt-get install cmake gcc-c++ qt5-base-devel qt5-tools-devel qt5-base-common boost-devel
    Источник
  2. Подготовить каталог сборки и запустить CMake:
    mkdir -p build
cmake -B build .
    Источник
  3. Выполнить сборку:
    cd build && make -j `nproc`
    Источник

Результирующие бинарные файлы можно разместить в каталоге, из которого их подхватывает инфраструктура Alterator, либо упаковать в RPM-пакет.

2.3. Типы значений и локализация

Спецификация поддерживает типы string, localestring, iconstring, stringlist, command, boolean, numeric и определяет правила их применения.Источник

  • string — ASCII-строка без управляющих символов; подходит для идентификаторов.Источник
  • localestring — текст, отображаемый пользователю; обязательно кодирование UTF-8.Источник
  • iconstring — имя иконки или путь к файлу, совместимый со спецификацией тем значков Freedesktop.Источник
  • stringlist — список строк, разделённых пробелами; используется для перечислений.Источник
  • command — команда shell, выполняемая модулем (например executor).Источник

Локализация значений выполняется через ключи вида display_name.ru = и display_name.en = внутри соответствующих групп. Необходимо обеспечивать полный набор языковых меток для интерфейсов.Источник

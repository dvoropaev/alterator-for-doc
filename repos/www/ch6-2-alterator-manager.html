6.2. Каталог alterator-manager

Каталог repos/alterator-manager содержит реализацию системной службы alterator-manager, исходные файлы C и документацию по формату backend.Источник

Важные элементы:

  • docs/README-ru.md — описание формата backend-файлов, структуры данных и процесса валидации.Источник
  • alterator_manager_*.c — исходные файлы, реализующие работу с D-Bus и polkit.Источник
  • org.altlinux.alterator-manager.service — unit-файл systemd для запуска службы.Источник

При сопровождении службы следует опираться на документацию из каталога docs/ для поддержания совместимости backend-описаний.

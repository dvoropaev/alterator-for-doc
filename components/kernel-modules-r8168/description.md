Linux driver for RealTek Ethernet controllers.

RTL8168 is a Linux device driver released for RealTek Gigabit Ethernet controllers with a PCI-Express interface, including the following models:

* RTL8168B / 8111B
* RTL8168C / 8111C
* RTL8168CP / 8111CP
* RTL8168D / 8111D
* RTL8168DP / 8111DP
